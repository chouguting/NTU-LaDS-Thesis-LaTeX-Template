# LaDS 論文模板

## 簡介 (Short Description)
本論文模板提供LaDS的學弟妹們一個撰寫碩士論文的範本，包含了基本的格式設定與範例內容，方便快速上手撰寫論文。

## 撰寫指南 (Writing Guide)
1. 請先配置好LaTeX環境，可以再Windows上安裝環境並使用VSCODE編輯器進行撰寫。
    - 安裝方式可以參考[這篇文章](https://hackmd.io/@HankTsai/H1N2n3zAa)
    - 注意事項: 若安裝完成後一直無法編譯，可能會需要重新開機一次。
2. 將本專案下載到本地端，並使用VSCODE開啟資料夾。
3. 修改 `ntusetup.tex` 檔案中的個人資訊，如論文標題、姓名、學號、指導教授等。
4. 在 `front/abstract.tex` 中撰寫中文與英文摘要。
5. 在 `chapters/` 資料夾中撰寫各章內容。
6. 每次儲存時，VSCODE會自動編譯並產生PDF檔案，請確認內容無誤。 (產生的PDF檔案會叫做main.pdf)
